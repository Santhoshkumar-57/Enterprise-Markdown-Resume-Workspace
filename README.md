# Enterprise-Markdown-Resume-Workspace
Markdown Resume Workspace is a 1-file web app turning Markdown into print-ready resumes. Features live split-screen editing, 3 industry themes, an AI tailoring hub, local auto-save, and pixel-perfect PDF export via browser print. Built with HTML, Tailwind, and JS, it requires zero setup or server installations.


A single-file web application that transforms plain Markdown into professional, print-ready resumes in real time. 

##  Features
- **Live Split-Screen:** Markdown editor on the left, high-fidelity A4/Letter canvas preview on the right.
- **3 Theme Layouts:** Swap instantly between *Executive Corporate*, *Tech Minimalist*, and *Creative Modern*.
- **AI Optimization Hub:** A "Job Description Matcher" panel to catch keyword gaps and generate tailored AI prompts.
- **Flawless PDF Export:** Custom `@media print` CSS strips the UI away for pixel-perfect browser printing (`Ctrl+P`).
- **Zero Setup:** Self-contained in a single `.html` file. No `npm install`, servers, or build steps required.

##  Quick Start
1. Download or copy the `index.html` file.
2. Double-click the file to open it in any modern browser.
3. Write your content, select a theme, and hit **Export to PDF**.

##  Tech Stack
- HTML5 & Vanilla ES6+ JavaScript
- Tailwind CSS & Marked.js (via CDN)
